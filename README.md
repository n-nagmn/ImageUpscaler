# Image Upscaler

ブラウザ上で動作する、アニメやイラストに特化した画像拡大（アップスケール）ツールです。
完全なローカル処理（クライアントサイド処理）で実行されるため、外部のサーバーへ画像が送信されることはなく、プライバシーを保ったまま安全に利用できます。

## 特徴

- **アニメ・イラスト特化モデル**: 主線やベタ塗りの境界を綺麗に保つ「ESRGAN-Thick」モデルを採用しており、高品質な拡大が可能です。
- **独立したノイズ除去機能**: 拡大処理の前に「Maxim Denoising」モデルを使用し、JPEG圧縮特有のモスキートノイズやブロックノイズを完全に除去してから再構築を行います。
- **プライバシー保護**: すべてのAI処理はブラウザの内部（オフライン状態に近い形）で完結します。
- **GPUアクセラレーション対応**: PCに搭載されている独立GPU（NVIDIAやRadeonなど）を優先的に使用する設定が組み込まれており、処理の高速化を図っています。

## 使い方

1. `index.html` を最新のWebブラウザ（Google Chrome, Microsoft Edgeなど）で開きます。
2. **画像選択**: アップスケールしたい画像（PNG, JPEG, WebP形式）を選択します。
3. **ノイズ除去**: 元画像に粗やノイズが目立つ場合は「実行する」を選択します（※AI処理が2段階になるため、計算時間は長くなります）。
4. **アップスケール倍率**: `2x`（2倍）または `4x`（4倍）から選択します。
5. **画像形式**: 保存時のフォーマットを `PNG` または `WebP` から選択します。
6. **実行**: 
   - `ページに表示`: 処理完了後、画面下部に結果を表示します。
   - `保存`: 処理完了後、自動的にパソコンへ画像をダウンロードします。

## 動作環境

- **推奨ブラウザ**: Google Chrome, Microsoft Edge (最新版)
- **必須要件**: ブラウザの「ハードウェアアクセラレーション（WebGL）」が有効になっていること。
- **推奨スペック**: AIによる重い画像処理を行うため、専用のグラフィックボード（dGPU）を搭載したPC環境を推奨します。

## 使用している技術・ライブラリ

本プログラムは、CDN経由で以下のオープンソースライブラリを読み込んで動作しています。

- [TensorFlow.js](https://www.tensorflow.org/js) - ブラウザ向け機械学習ライブラリ
- [UpscalerJS](https://upscalerjs.com/) - ブラウザ向け超解像ライブラリ
- 使用モデル:
  - `@upscalerjs/maxim-denoising` (ノイズ除去用)
  - `@upscalerjs/esrgan-thick` (アニメ・イラスト拡大用)

## 注意事項

- 高解像度の画像を4倍に拡大しようとするなど、ビデオメモリ（VRAM）を大量に消費する処理を行うと、ブラウザのタブがクラッシュする可能性があります。
- AIによる処理のため、画像のサイズやPCのスペックによっては完了までに数十秒〜数分程度の時間がかかる場合があります。
